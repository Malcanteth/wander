--------------------------
Список изменений в версиях
--------------------------
0.11 (25/01/2010)
- доработаны трупы от противников, теперь они наследуют некоторые характеристики от живого монстра
- если герой стоит на месте, где уже лежит предмет, он все-равно его сможет выкинуть, если рядом с ним есть для этого место
- после смерти выбрасываются вещи героя (и монстра, но пока они генерятся без предметов)
- можно сделать скриншоты (F5)
- теперь нужно выбрать пол и ввести имя персонажа 
- появились секретные двери
- добавлены команды 'Открыть' и 'Осмотреться'
- если денег на полное лечение не хватает - целительница сможет подлечить и за сумму, которая имеется
- каждые три уровня герой может увеличить одну из своих характеристик
- добавлены "Способности", пока их всего 7, но все они уже действуют
- немного доработана битва (контратака, несколько ударов подряд)
- увеличено окно на две строчки для вывода сообщений
- ну и как всегда несколько мелких изменений

0.10a (04/01/2010)
Изменение только одно:
- думаю, что наконец расправился с багом клонирования героя! 

0.10 (22/12/2009)
- функция "Отдать" теперь работает - пока она нужна только для полного завершения квеста
- теперь не нужно в ручную убирать предмет из экипировки, если нужно использовать другой предмет
- немного подправлен "баланс"
- некоторые другие мелкие изменения
Исправлены ошибки:
- серьезная ошибка с клонированием героя и атакой невидимок
- несколько мелких ошибок

0.09 (20/12/2009)
- в деревню добавлены целительница и мясник
- щит теперь не действует как броня, зато им можно отбить атаку
- есть и другие более мелкие изменения
Исправлены ошибки:
- при подъеме по лестнице герой появляется не там, где надо (Харука-тян)
- ГГ не умирал от голода (Sanja)
- при выкидывании/съедении предмета вес не уменьшается (Харука-тян)
